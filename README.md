# Spellstacks

A daily word puzzle game. Use all 18 letters to make words and solve the puzzle in as few words as possible.

Play at [spellstacks.com](https://spellstacks.com)

## How to Play

1. Select letters from the rack to spell words
2. Add valid words to your stack
3. Use all 18 letters to complete the puzzle
4. Try to finish in as few words as possible

## Features

- New puzzle every day
- Track your stats and streaks
- Share your results with friends
- Works on desktop and mobile
- Colorblind-friendly design

## Development

This is a static web app with no build step required. To run locally:

1. Clone the repository
2. Serve the files with any static web server
3. Open in your browser

Example using Python:
```bash
python -m http.server 8000
```

## Tech Stack

- Vanilla HTML, CSS, JavaScript
- No frameworks or dependencies
- Local storage for game state and stats
